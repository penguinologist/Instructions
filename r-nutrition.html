LibreOffice provides a great variety of tools you need to get work done. One of these is LibreOffice Calc. Similar to Microsoft Excel, this program will help you create spreadsheets and organize your data.

Pre-requisites

  • An installation of LibreOffice
  • data to be entered
  • time

Steps

  1. open LibreOffice Impress
  2. follow the onscreen instructions where to add a title and text
  3. to add a new slide, right-click in the left pane and select “New slide”
  4. repeat steps 2 and 3 until all text is entered
  5. click the File menu in the top left corner
  6. click save
  7. proceed to pick a place where to save and give the document a name.
  8. Click save
  9. finished
