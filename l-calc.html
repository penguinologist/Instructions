LibreOffice provides a great variety of tools you need to get work done. One of these is LibreOffice Calc. Similar to Microsoft Excel, this program will help you create spreadsheets and organize your data.

Pre-requisites

  • An installation of LibreOffice
  • data to be entered
  • time

Steps

  1. 1. open LibreOffice Calc
  2. 2. proceed to enter data in the fields
  3. 3. click the File menu in the top left corner
  4. 4. click save
  5. 5. proceed to pick a place where to save and give the document a name.
  6. 6. Click save
  7. 7. finished
